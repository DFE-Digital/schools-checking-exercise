Terminal or critical illness

Design changes

Change: Removed 'Has John Smith been under investigation for a serious or life changing illness for 12 months or more?'

Reason: We do not need to know this information as per the guidance. We have to ask the user if they've had a critical illness for longer than 12 months, and we cover this in a previous question.

Severity: High

Flow changes

! Warning If routing is not going to be implemented, follow this flow without it instead.

Change: 'Has this had a severe and profound effect on their ability to access education or sit exams?' is only asked if the user answers yes to either

  • 'Have they recently been diagnosed with a life changing illness?', or
  • 'Have they recently been diagnosed with a life changing injury?'

Reason: According to the guidance and current questions these are the only times we need to know if the pupil has been impacted in a severe way. This change means we can ask the users only relevant questions and not increase their cognitive load unnecessarily.

Implementation: Link to implemented flow.

Severity: Medium

Recommendations

Applies to: Flow of the questions

Recommendation: Change the flow so the user can answer the minimum amount of questions to get to their summary and evidence page.

This means that when the user answers Yes to questions that fulfil a removal criteria they skip the remaining questions. For example,

  • 'Have they been diagnosed with a terminal illness? Yes or No'
  • Yes - 'Have they sat any year 11 exams? Yes or No'
  • Journey goes to Summary, then Evidence.

Reason to change: The user has to answer every question in the flow even if they fulfil the first case of the pupil being diagnosed with a terminal illness. This could cause stress and worry that the pupil does not fulfil the right criteria.
